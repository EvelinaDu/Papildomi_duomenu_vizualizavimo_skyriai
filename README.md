# Advanced Data Visualization

This repository contains university laboratory assignments and algorithmic implementations focused on statistical data analysis, dimensionality reduction, and complex data visualization.

---

## Projects

### 1. Loan Granting Decision Analysis (Lab 1)
**Directory:** `1_laboratorinis/`

[![R](https://img.shields.io/badge/R-%23276DC3.svg?style=flat&logo=r&logoColor=white)](https://www.r-project.org/) [![Tidyverse](https://img.shields.io/badge/Tidyverse-%23276DC3.svg?style=flat&logo=r&logoColor=white)](https://www.tidyverse.org/) [![GGally](https://img.shields.io/badge/GGally-blue?style=flat)](https://ggobi.github.io/ggally/) [![Corrplot](https://img.shields.io/badge/Corrplot-blue?style=flat)](https://github.com/taiyun/corrplot)

A detailed investigation into the factors influencing loan approval using LendingClub historical data.
- **Data Engineering:** Outlier detection using Interquartile Range (IQR) methods.
- **Statistical Analysis:** Evaluation of borrower risk profiles through FICO scores, Debt-to-Income (DTI) ratios, and annual income distributions.
- **Visual Analytics:** Development of stratified visualizations to identify non-linear relationships between borrower characteristics and credit policy adherence.

### 2. Principal Component Analysis (PCA) Framework
**Directory:** `Algoritmas/`

[![Python](https://img.shields.io/badge/python-3670A0?style=flat&logo=python&logoColor=ffdd54)](https://www.python.org/) [![Pandas](https://img.shields.io/badge/pandas-%23150458.svg?style=flat&logo=pandas&logoColor=white)](https://pandas.pydata.org/) [![NumPy](https://img.shields.io/badge/numpy-%23013243.svg?style=flat&logo=numpy&logoColor=white)](https://numpy.org/) [![scikit-learn](https://img.shields.io/badge/scikit--learn-%23F7931E.svg?style=flat&logo=scikit-learn&logoColor=white)](https://scikit-learn.org/) [![Matplotlib](https://img.shields.io/badge/Matplotlib-%23ffffff.svg?style=flat&logo=Matplotlib&logoColor=black)](https://matplotlib.org/)

A comprehensive study of dimensionality reduction techniques, covering both classical and modern PCA variations.
- **Algorithmic Foundation:** Detailed implementation of PCA via Singular Value Decomposition (SVD) and Eigen-decomposition of covariance matrices.
- **Advanced Variants:** Theoretical and practical exploration of Kernel PCA (non-linear), Sparse PCA (interpretability), and Robust PCA (outlier sensitivity).
- **Interpretation Tools:** Utilization of Scree plots for variance analysis, Loading Score plots for feature contribution, and Biplots for multivariate observations.
